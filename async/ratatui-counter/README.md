# ratatui-counter

Counter application with ratatui-async-template

## Run demo

```rust
export RATATUI_COUNTER_CONFIG=`pwd`/.config
export RATATUI_COUNTER_DATA=`pwd`/.data
export RATATUI_COUNTER_LOG_LEVEL=debug

cargo run
```
